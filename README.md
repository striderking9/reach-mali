# Reach Mali MSNA Scripts [ Cleaning & Analysis ]

Ce repos contient les scripts, kobo et checklist utilisé lors du msna 2025.

- Le fichier R contient les scripts de nettoyage des données ménage ainsi que les données individuelle des ménages.

- Les fichier check_list et kobo contiennent respectivement le formulaire kobo ainsi que les check_list.

J'ajouterai une base de données nettoyées que tu peux utiliser pour tester les scripts. 
J'éviterai d'ajouter les bases brutes vu qu'elles sont assez sensible.

Les packages à installer comportent quelques packages basique ainsi que des packages mis en place par le siège pour faciliter la construction des scripts.

Le lien ci-dessous comporte également des packages mis en place par le siège pour faciliter la construction des scripts.
https://github.com/impact-initiatives/cleaningtools

Le lien ci-dessous comporte les packages d'analyses mis en place par le siège qu'on utilise pour faire des analyses préliminaires. 
https://github.com/impact-initiatives/analysistools

Le package presentresult ci-joint est également un package du siège qui nous permet de bien restructurer la sortie des données sur Excel.
https://github.com/impact-initiatives/presentresults

Le package ci-joint est conçu pour créer des rapports de contrôle qualité, nettoyer, analyser et présenter les résultats des indicateurs de résultats clés de l'unité de santé publique. 
Il est également intégré dans nos vérification et analyse afin d'ajouter des indicateurs rapidement conformément au directive du siège.
https://github.com/impact-initiatives/impactR4PHU
