# Reach Mali MSNA Scripts

Ce repos contient les scripts, kobo et checklist utilisé lors du msna 2025.

- Le fichier R contient les scripts de nettoyage des données ménage ainsi que les données individuelle des ménages.

- Les fichier check_list et kobo contiennent respectivement le formulaire kobo ainsi que les check_list.

J'ajouterai une base de données nettoyées que tu peux utiliser pour tester les scripts. 
J'éviterai d'ajouter les bases brutes vu qu'elles sont assez sensible.

Les packages à installer comportent quelques packages basique ainsi que des packages mis en place par le siège pour faciliter la construction des scripts.

Le lien ci-dessous comporte également des packages mis en place par le siège pour faciliter la construction des scripts.
